# Data directory

Main files:

- `final_analysis_data_with_bti_and_selection.csv`: harmonized pre-Q1 input dataset used by `reproduce_q1_2026_analysis.py`.
- `AI_Diffusion_Q12026_Update.csv`: Microsoft Q1 2026 update file.
- `final_analysis_data_q1_2026.csv`: generated Q1 2026 analysis dataset.
- `final_analysis_data_q1_2026_with_alternative_gaps.csv`: generated dataset with alternative access-baseline residuals.
- `final_analysis_data_q1_2026_with_wb_metadata.csv`: generated dataset with World Bank region and income group metadata.
- `data_dictionary.csv`: generated variable dictionary for the Q1 2026 analysis dataset.

The included data contain public country-level indicators and no personal data.
