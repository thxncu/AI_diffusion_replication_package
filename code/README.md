# Code directory

- `00_run_all.py`: runs the complete workflow.
- `reproduce_q1_2026_analysis.py`: merges the Q1 2026 Microsoft update, reconstructs k = 2 and k = 3 profiles, estimates main regressions, and generates main figures and tables.
- `reproduce_extended_robustness_analysis.py`: runs the extended robustness checks used in the Supplementary Material.

Run from repository root:

```bash
python code/00_run_all.py
```
