# GitHub upload notes for journal submission

1. Create a new repository and upload the full contents of this package.
2. For double-anonymized review, avoid a repository name, account name, commit history, or profile metadata that identifies the author. An anonymized Open Science Framework link or an anonymous GitHub workflow can also be used.
3. Keep the repository private or anonymized during review if required by the journal.
4. After acceptance, replace anonymous placeholders in `CITATION.cff`, add author details, and archive a release on Zenodo, OSF, or Mendeley Data if desired.
5. Include the repository link in the manuscript data availability statement.

Suggested data availability wording:

> The replication package, including harmonized country-level data, analysis code, generated tables, and figure scripts, is available at [repository link]. During double-anonymized review, an anonymized repository link is provided.
